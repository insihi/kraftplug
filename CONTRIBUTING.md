# Contributing

Bug reports and pull requests are welcome.

Before opening an issue:

1. Update to the latest release.
2. Restart Home Assistant.
3. Check the Home Assistant logs for `custom_components.kraftplugg`.
4. Download diagnostics from the KraftPlugg device page.

Never post phone numbers, SMS codes, access tokens, refresh cookies, addresses,
or complete meter identifiers.

For code changes, keep the integration asynchronous, avoid blocking I/O, update
translations when UI text changes, and run the HACS and Hassfest validations.
